# 项目名称
博客 （雏形）  

# 功能
使用脚本化的方式，使用Vue做一个博客练练手：  
1、发表博文、发表评论。  
2、博文列表、博文详细、讨论列表。  
 
# 内容
一个简单的单人博客。  
# 文件说明
  
 index.html 博客页面，单页的。  
  
 css 文件夹里面是需要的css文件。俺css比较烂，所以用的以前项目的css，文件有点多。  
# 安装要求
需要有vue.js文件，这里有提供。  
需要谷歌内核浏览器，IE还是算了，不支持。 
后端暂时是asp.net的，打算改成node

# 使用方式
直接运行即可，不用建立站点，也不用各种配置，简单方便。  
没有后端跑不起来，打算先弄个json文件凑合一下。
